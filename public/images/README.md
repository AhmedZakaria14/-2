# Images Directory

This directory contains all the images for the website. Please upload your images to the respective folders.

- `hero/`: Hero section images (e.g., hero-main.jpg)
- `painting/`: Painting service images
- `gypsum/`: Gypsum service images
- `epoxy/`: Epoxy service images
- `shelters/`: Shelters service images
- `renovation/`: Renovation service images
- `wallpaper/`: Wallpaper service images
- `tiles/`: Tiles service images
- `portfolio/`: Portfolio gallery images
- `blog/`: Blog post images
- `icons/`: Custom icons
- `logo.png`: Website logo
- `og-image.jpg`: OpenGraph image (1200x630)
